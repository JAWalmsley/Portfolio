![](/images/ENC_home.png)

Users can upload any image to encrypt it with a given password using AES-128 encryption. The encrypted image is then downloaded as a .enc file which can be stored on the computer. The .enc files can be decrypted later, back to the original file.

![](/images/ENC_encrypt.png)


## Built with
* Django
* Python's `cryptography` module (`Fernet`, `PBKDF2HMAC`)
* HTML/CSS
* Materialize-css
